How to Create a GitHub Repository

  1. Go to GitHub.com and log in to your account.
  2. Click the "New" button or “+” icon at the top right.
  3. Enter a repository name and description (optional).
  4. Choose Public or Private.
  5. Click the Create repository button.

How to Make a Repository Live using GitHub Pages

  1. Go to your repository on GitHub.
  2. Click on the Settings tab.
  3. Scroll down to the Pages section.
  4. Under Source, select the branch (usually main), and then click Save.
  5. After a few seconds, your site will be live at:
    https://your-username.github.io/your-repo-name
